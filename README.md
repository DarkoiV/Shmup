## Controls
```
Arrows  - movement  
z       - shot  
Shift   - focus mode  
```

## Building and running on Linux
Make sure you have ` CMake(3.22 or newer), SDL2 and SDL2_image.  `

```bash
mkdir build; cd build 
cmake ..
make
cd ..
./shmup
```
